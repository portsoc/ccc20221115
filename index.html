Hello Mum :-)

There are three factors that affect the style of this paragraph. Firstly there is the browser's default stylesheet which sets margins above and below every paragraph. Then there is a CSS rule with a p selector, which sets the background to yellow and adds padding around the text. Next (because the p element has a class attribute with values of 'important' and 'cool'), the important class is applied which overrides the background color making it red, and setting the text to white. Finally the cool class is applied which changes the font family and makes the text bold.
